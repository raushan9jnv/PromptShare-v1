# PromptShare V2 Changes

## What changed in this version
This version is a major creator-first redesign.

### Main product changes
- homepage restructured around creator use cases
- seeded dummy prompts added so the site feels alive before the DB is full
- sidebar simplified into a premium browse rail
- prompt detail page reorganized around visual result story first
- image viewer improved with input/output-aware media handling
- submit page rebuilt into a guided creator upload flow
- browse pages updated to match the new premium system

### Visual changes
- kept the warm orange editorial palette as the standard
- kept Fraunces + Inter + JetBrains Mono as the font system
- improved spacing, rounded surfaces, and premium card rhythm
- reduced messy layout behavior on prompt pages and navigation

### Content changes
- stronger creator focus
- image transformations and social systems now lead the product story
- software/study prompts still exist, but live more naturally in secondary lanes

### Technical changes
- seeded prompt content now merges with live prompt data
- prompt asset UI supports media roles in the frontend
- homepage collections can be driven from seeded content until live content grows

## Why this matters
The previous version had the right idea but still felt messy in important places.
This version is meant to make the site feel immediately impressive, useful, and more trustworthy for real creator workflows.
